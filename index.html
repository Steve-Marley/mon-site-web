Fonctionnalités

Ce chatbot a de nombreuses fonctionnalités intégrées pour offrir une interaction riche avec les utilisateurs. Voici un aperçu détaillé des fonctionnalités disponibles :

Système de Connexion et d'Inscription

Les utilisateurs peuvent se connecter ou s'inscrire via l'interface de connexion. Le système vérifie les informations de l'utilisateur dans la base de données SQLite. Une interface utilisateur permet l'inscription de nouveaux utilisateurs avec des champs pour le prénom, le nom, l'email et le mot de passe.

Fenêtre Principale

Après la connexion, les utilisateurs accèdent à une fenêtre principale où ils peuvent choisir différentes options :

  • Chat : Permet aux utilisateurs d'entrer dans une fenêtre de chat pour discuter avec le chatbot.
  • Assistant Personnel : Fournit une interface pour des interactions vocales avec le chatbot.
  • À propos : Affiche des informations sur l'application.
  • Réglages : Permet de modifier les paramètres de l'application.
  • Historique : Affiche l'historique des conversations des utilisateurs.

Chat

Interface de chat textuelle où les utilisateurs peuvent envoyer des messages au chatbot. Le chatbot répond aux messages des utilisateurs en utilisant des paires de réponses prédéfinies et peut aussi effectuer des recherches sur Google pour des réponses supplémentaires. Les conversations sont stockées dans une base de données SQLite avec des horodatages.

Assistant Personnel

Interface pour des interactions vocales avec le chatbot. Utilise speech_recognition pour la reconnaissance vocale et pyttsx3 pour la synthèse vocale. Le chatbot peut répondre aux commandes vocales pour :

  • Dire bonjour.
  • Répondre à des questions sur l'heure.
  • Jouer des chansons sur YouTube via pywhatkit.
  • Répondre à des questions basiques sur son nom.

Réglages

Interface pour modifier divers paramètres de l'application :

  • Changer la voix de l'assistant (français, anglais, espagnol).
  • Régler le volume.
  • Changer la police d'affichage.
  • Changer la langue de l'interface.
  • Changer le fond d'écran.

Historique

Interface pour afficher l'historique des conversations de l'utilisateur. Les conversations sont récupérées à partir de la base de données SQLite et affichées avec les horodatages.

Documentation du Code Python

ChatbotThread (Classe)

Cette classe gère la création d'un thread pour le chatbot et l'exécute en répondant aux messages ou en effectuant une recherche Google si aucune réponse n'est trouvée dans les paires prédéfinies.

  • response_ready: Signal émis lorsqu'une réponse est prête.
  • run(): Méthode principale du thread, utilise le module nltk pour répondre aux messages ou recherche sur Google.
  • google_search(query): Effectue une recherche Google en utilisant BeautifulSoup et retourne la réponse trouvée.

DatabaseManager (Classe)

Cette classe gère la connexion à la base de données SQLite (clients.db) et fournit des méthodes pour insérer des clients et récupérer leur historique.

  • __init__(): Initialise la connexion à la base de données et crée les tables si elles n'existent pas.
  • create_tables(): Crée les tables clients et history si elles n'existent pas.
  • insert_client(firstname, lastname, email, password): Insère un nouveau client dans la table clients.
  • get_client(username): Récupère le prénom et le mot de passe d'un client à partir de la table clients.
  • insert_history(user_name, timestamp, conversation): Insère une entrée d'historique dans la table history.
  • get_history(user_name): Récupère tout l'historique des conversations pour un utilisateur donné.

WindowManager (Classe)

Cette classe gère les fenêtres de l'application PyQt5 et coordonne le passage entre la fenêtre d'inscription et la fenêtre principale du chatbot.

  • __init__(): Initialise l'application PyQt5, crée une instance de DatabaseManager et affiche la fenêtre d'inscription.
  • show_registration_window(): Affiche la fenêtre d'inscription.
  • show_chatbot_window(user_name): Affiche la fenêtre principale du chatbot après la connexion.
  • run(): Exécute l'application PyQt5.
